Dataset: Animal Image Dataset 151 classes

Dataset Animal Image Dataset 151 classes dari Kaggle berisi lebih dari 26.000 gambar hewan dalam 10 kelas yang dikumpulkan dari Google Images. Dataset ini banyak digunakan sebagai benchmark untuk klasifikasi gambar menggunakan CNN.

Arsitektur Model CNN — MobileNetV2 + Custom Head

Model menggunakan Transfer Learning dengan base model MobileNetV2 yang telah dilatih pada dataset ImageNet (1.4 juta gambar, 1000 kelas). Layer-layer di base model dibekukan saat fase awal training, kemudian di-fine-tune pada fase kedua.
